# 🔍 PhishIris – Phishing Detection & Threat Analysis Dashboard

PhishIris is a web-based phishing detection tool that simulates SOC workflows by analyzing emails, extracting IOCs, and classifying threats using risk scoring and threat intelligence.

---

## 🚀 Features

- Email header & content analysis (SPF, DKIM, DMARC)
- Phishing detection using rule-based engine
- IOC extraction (IPs, domains, URLs)
- VirusTotal integration for threat intelligence
- Risk classification (Low / Medium / High / Critical)
- Dashboard visualization with indicators & recommendations

---

## 🛠️ Tech Stack

- Python (Flask)
- HTML, CSS, JavaScript
- VirusTotal API
